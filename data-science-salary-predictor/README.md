# Data Science Salary Predictor

## Overview

This project analyzes global data science salary trends and predicts salaries based on professional and company-related factors. It combines exploratory data analysis, machine learning models, and an interactive Streamlit application to provide salary insights and predictions.

The project demonstrates the complete data science workflow, from data preprocessing and visualization to model building and deployment.

## Features

* Interactive salary analysis dashboard
* Exploratory Data Analysis (EDA)
* Salary prediction using machine learning
* Multiple regression models for comparison
* Feature importance analysis
* Real-time predictions through Streamlit
* Interactive visualizations with Plotly

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* Streamlit
* Plotly
* Matplotlib
* Seaborn

## How It Works

1. Salary data is cleaned and preprocessed.
2. Features are encoded and prepared for modeling.
3. Multiple regression models are trained and evaluated.
4. The best-performing models are used for prediction.
5. Users provide profile details through the Streamlit interface.
6. The application generates salary estimates and visual insights.

```bash

```

## Run the Project

```bash

```

The application will be available at:

```text
http://localhost:8501
```
